
This disc can run on Macintosh, Windows, Unix and Linux computers
with a Web browser.


INSTRUCTIONS FOR MACINTOSH USERS


To read this disc on a Macintosh,
double-click the "Index.htm" file. Your operating
system should launch the default Web browser.
If nothing happens, try and launch your favorite Web browser,
and then click "Open File" in the "File" menu 
and select the "Index.htm" file at the root of the disc.


INSTRUCTIONS FOR WINDOWS USERS

If the "auto insert notification" is enabled on your PC, then the disc
will launch automatically. Then follow the instructions displayed on the screen.
You can also open Windows Explorer, right-click on the CD/DVD drive, and then
select "AutoPlay".
